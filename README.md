# Sales Data Cleaning Project (Power Query)

## Project Overview
This project focuses on cleaning and preparing a raw sales dataset using Microsoft Excel Power Query. The goal was to transform messy raw data into a structured format suitable for analysis.

## Dataset Information
The dataset contains sales transaction records including:

- Transaction Date
- Stock Date
- Product ID
- Customer ID
- Store ID
- Quantity Sold

Total Records: 182,883  
Total Columns: 6

## Tools Used
- Microsoft Excel
- Power Query

## Data Cleaning Steps
The following data cleaning steps were performed using Power Query:

1. Imported raw dataset into Power Query
2. Checked and removed duplicate records
3. Verified and corrected data types
4. Converted date columns into proper date format
5. Renamed columns for better readability
6. Checked for missing values
7. Loaded the cleaned data back to Excel

## Final Output
The cleaned dataset is structured and ready for further analysis such as:

- Sales Analysis
- Product Performance
- Store Performance
- Customer Purchasing Patterns
